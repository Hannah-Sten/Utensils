# Utensils

## Gradle dependency
Add the following to your root `build.gradle`.

```
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
```
dependencies {
    compile 'com.github.Ruben-Sten:Utensils:master-SNAPSHOT'
}
  ```
